Sistema de Caja para Kiosco - Unidad 5
Este proyecto es la resolución del Trabajo Práctico de la Unidad 5 de Programación Orientada a Objetos para el IES Nº 6001. Se trata de un sistema de facturación, control de stock y reportes de recaudación desarrollado en Python, estructurado bajo el patrón Repositorio y utilizando SQLite para la persistencia de los datos.

Para probar la versión principal por consola, debes ejecutar el archivo main.py en tu entorno. 

Tambien a su vez disponemos de una version con una simple interfaz gráfica, ejecuta main_gui.py. El sistema se encarga de generar automáticamente la base de datos necesaria al iniciar, por lo que no requieres configuraciones manuales previas.
